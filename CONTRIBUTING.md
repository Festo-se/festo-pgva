# Contributing to festo-pgva

Thank you for your interest in contributing to festo-pgva! We are a lean team and are grateful for contributions from the community. 

Did you find a bug?
- If you have found a security-related bug, please do NOT use the Github Issue tracker to submits. Instead, pelase refer to our Security guidelines found [here](SECURITY.md). 

- Before reporting non-security bugs, please search the [issue tracker](https://github.com/Festo-se/festo-pgva/issues) to see if your issue has already been reported. 

- If your bug has not been reported, please [open a new issue](https://github.com/Festo-se/festo-pgva/issues/new). Include a title, a clear and concise summary of the bugged behavior, as much contextual information for your operational set-up as possible, and code examples demonstrating the bug.

## New Contributors

Want to contribute to the codebase itself? Please get started at the README to set up your clone of our repo, make a fork, and make a pull request from your fork with your changes for us to review!

- [Setting up Git](https://docs.github.com/en/get-started/git-basics/set-up-git) 

- [Pull requests](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests)
